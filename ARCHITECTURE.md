# Chronicle — Полная архитектура

## Философия

Модульное ST-расширение для глубокого отслеживания RP-мира.  
**Гибрид**: AI выдаёт данные в тегах → код парсит, валидирует, применяет.  
**Стиль**: glassmorphism, FA-иконки, адаптивность.  
**Язык UI**: русский.

---

## Формат AI-тегов (полный справочник)

### `<world>` — КАЖДЫЙ ход

```
time:2026/3/15 14:30
location:Квартира·Кухня
weather:пасмурно, мелкий дождь
atmosphere:уютная, запах кофе
characters:Даниил,Лиза
costume:Даниил=домашние штаны, без футболки
costume:Лиза=оверсайз свитер, шорты
```

### `<event>` — КАЖДЫЙ ход

```
важное|Даниил приготовил завтрак и обжёг руку
```

### `<sims>` — КАЖДЫЙ ход

```
hunger:65|ел яичницу
hygiene:80
sleep:40|не спал всю ночь
arousal:15
```

### `<health>` — при изменениях

```
hp:95
intoxication:30|два бокала вина
injury:ожог правой руки|лёгкий
habit:курение|3 сигареты
```

### `<cycle>` — при изменениях

```
day:14
phase:овуляция
symptoms:лёгкие спазмы
```

### `<diary>` — когда есть что записать

```
Даниил|Она снова не ела. Надо поговорить.
{{user}}|Он подозрительно ласковый.
```

### `<wallet>` — при транзакциях

```
balance:45000₽
spend:еда|850₽|продукты
income:5000₽|перевод от мамы
```

### `<npc>` — при появлении / изменении

```
Лиза|внешность=рыжие волосы|характер=вспыльчивая|отношение=подруга|пол:ж|возраст:23|birthday:1995/6/12
```

### `<agenda>` / `<agenda->` — задачи

```
2026/3/25|День рождения Лизы
```

### `<location>` — описание + связи комнат

```
Квартира·Кухня|описание=маленькая, окно во двор|связь=Квартира·Прихожая,Квартира·Гостиная
```

### `<item>` / `<item->` — предметы

```
item:🔑Ключи от квартиры|Даниил@карман куртки
item-:Сигареты
```

### `<affection>` — симпатия NPC к {{user}}

```
Даниил=+5|помогла с ужином
```

## Вкладки Drawer (9 вкладок)

1. **Статус** `fa-gauge` — обзор: время, погода, локация, наряды, мини-симс
2. **Симс** `fa-heart-pulse` — 4 бара + история + пороги
3. **Здоровье** `fa-kit-medical` — HP, опьянение, травмы, привычки, цикл
4. **Таймлайн** `fa-timeline` — лента событий
5. **Персонажи** `fa-users` — NPC карточки + ДР + симпатия + дневник
6. **Предметы** `fa-box-open` — инвентарь + кошелёк/бюджет
7. **Карта** `fa-map` — интерактивные блоки-комнаты
8. **Календарь** `fa-calendar-days` — месячная сетка + метки
9. **Настройки** `fa-gear` — модули, промпты, темы
